# TodoApp

This is a full-stack Todo application built with Django for the backend and React with TypeScript for the frontend.

## Features

- User registration and authentication
- JWT-based authentication
- Create, read, update, and delete todos
- Responsive design

## Backend

The backend is built with Django and Django REST Framework.

## Frontend

The frontend is built with React, TypeScript, and Vite.

## Usage

- Users can register and log in.
- Authenticated users can create, view, update, and delete their todos.
- Todos can be marked as completed, which will apply a line-through style.
- The application is responsive and works on various devices.
